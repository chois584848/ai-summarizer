# AI Summarizer

An interactive prototype of an AI summarization tool — summarize **YouTube videos, documents, websites, and text**.

Built as a single self-contained `index.html` (no build step). Open it directly or serve the folder.

## Features
- **4 source tabs**: YouTube · Document · Website · Text
- **YouTube**: real embeds + trending grid; click a timeline timestamp to jump to that moment in the video
- **Output formats**: Timeline · Key Points · Blog · TL;DR · Tweet thread · Email · Flashcards
- **Revise / Q&A chat** ("Mimi") grounded in the summary
- **Library** (Saved): searchable, filterable grid of saved summaries with a time-saved stat
- **Save flow** that adds items to your Library

> Prototype — summaries are mocked client-side.
